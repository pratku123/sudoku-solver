Sudoku

In a sudoku, we are given a partially filled 9 by 9 grid. The aim is fill up the empty cells such that, every row, column and 3 by 3 subgrid contains the digits 1 to 9 only once.

Sudoku solving algorithms


A few ways to solve the sudoku are as follows:
1. Backtracking
2. Brute Force
3. Stochastic Search
4. Constraint Programming

In this project, the back tracking algorithm has been implemented in Javascript with a bit of angular js.
